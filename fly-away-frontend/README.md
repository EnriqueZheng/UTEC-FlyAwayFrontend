# Fly Away – Frontend ✈️
## Requisitos

- Node.js 18+ y npm
- El **backend** corriendo en `http://localhost:8080`:
  ```bash
  # En la carpeta del backend
  ./mvnw spring-boot:run
  ```

## Cómo correrlo

```bash
npm install
npm run dev
```

Abre la URL que muestra Vite (por defecto `http://localhost:5173`).
